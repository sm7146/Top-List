Webpage - Text documents written in HTML language. They are intepretted by browsers and displayed.

*Computers are stupid, meaning they must be told what to do using SYNTAX.

Basic HTML Vocabulary

- Tags : Contained within angled brackets < > . Use opening and closing tags < / > that tell what action
is rquired by the tag and where to end the action. Some tags are considered VOID tags and do not
require a closing tag. Tags can be inline ( < b >, < em >, < span >) or block ( < p >, < div > ).

- Element : Refers to everyting with the opeing and closing tag.

- Attribute : Property of the HTML element. Ex. href is an attribute of the anchor tag < a href="www.udacity.com">
